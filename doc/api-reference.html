    Modules

    Truco

    Truco is a game developed in Elixir, as a TP for a class.
    It has several branches, this is the latest one.

    Truco.Card

    Structure that defines the layout of each card bieng used in the game.
    Each card will have a name, and a magic number (the "power" of the card).

    Truco.Game

    The more "generic" logic of a cards game is in here.
    The idea was to make a reusable logic, for applying in several games.

    Truco.Player

    Player structure, it defines the layout of each player in the game.

    Truco.Team

    This structure holds the attributes of a team.

    Truco.Turn

    Keeps the logic of the mayority of the game. It envolves the logic of a whole turn.
